# Work-Hive

## Description 

The aim of this project is to collaboratively create a full stack application that would allow the user to search for tech jobs around the world and also save these jobs to their profile and/or apply for different jobs. This application follwows the MVC paradigm in its architectural structure, using Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.

## Criteria

* Registration and Authentication: Users will be allowed to create accounts, login securely, and can manage their profiles.

* Employers can advertise job listings, including job title, description, salary, company name, job type and location.

* Job seekers with certain criteria can start a search with keywords such as job title, location and job type.

* Potential Job seekers can go through the platform and login, and can manage applications within their dashboard.

* Our platform is responsively Designed, and is accessible and user-friendly across different devices and screen sizes.


## Technical Criteria 

* Application uses a Node.js and Express.js back end and uses both GET and POST routes for retrieving and adding new data.

* Application has a folder structure that meets the MVC paradigm and uses Handlebars.js as the template engine.

* Application is backed by a PostgreSQL database with a Sequelize ORM and protects API keys and sensitive information with environment variables.

* Application includes user authentication (express-session and cookies).


## Usage 

* Once a user has navigated to the platform, they are able to create accounts to apply for or save jobs to their personal profile.

* On the homepage, users can search for jobs based on a number of criteria, including job type, title and location. 

* On the profile page, the user is able to view their CV as well as jobs they have applied for or saved to view at a later time.

* Candidates can apply for roles they are interested in effectively.

* Vacancies, location, salary details will be posted.

* Interview tips with links are to support there to support a succesful job search.
